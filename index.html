Welcome to the OBS counter widget!
This page will display a number for five seconds and hide it afterwards.
It can be useful for counting some events that occur during a stream.
Add this to OBS using the Browser source. After that, right-click the source and choose "Interact".
Left-click anywhere in the Interact window to increase the counter by 1.
Right-click to assign a custom value to the counter.
The counter value persists accross OBS restarts.
Do either of those now to hide these instructions. They will never be shown again.
Prefer a green-screen version, to use with a traditional browser? Add &green to the end of the URL.
Need multiple counters? Add &counter=some_name_here.
